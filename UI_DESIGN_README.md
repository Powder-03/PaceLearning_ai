## Design System

### Colors

| Name | Hex | Usage |
|------|-----|-------|
| Background | `#0f0a1a` | Main app background |
| Card/Input | `#1a1425` | Cards, form inputs, sidebar |
| Border | `#2d2640` | Card borders, dividers |
| Primary (AI Tutor) | `#8b5cf6` | Primary buttons, links, active states |
| Document Mode | `#3b82f6` | Document chat accent |
| Video Mode | `#22c55e` | Video learning accent |
| YouTube Mode | `#ef4444` | YouTube tutor accent |
| Coming Soon | `#6b7280` | Disabled/unavailable features |
| Text Primary | `#ffffff` | Headings, primary text |
| Text Muted | `#9ca3af` | Secondary text, labels |
| Success | `#22c55e` | Success states, completed |
| Error | `#ef4444` | Error states, validation |
| Warning | `#f59e0b` | Warning states |

### Typography

```css
/* Font Family */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* Headings */
h1: 32px, Bold, #ffffff
h2: 24px, Bold, #ffffff
h3: 20px, Semibold, #ffffff
h4: 16px, Semibold, #ffffff

/* Body */
body: 14px, Regular, #ffffff
body-small: 12px, Regular, #9ca3af

/* Labels */
label: 14px, Medium, #9ca3af
```

### Spacing

```
Base unit: 4px
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
2xl: 48px
```

### Border Radius

```
small: 4px (badges, small elements)
default: 8px (cards, inputs, buttons)
large: 12px (modals, large cards)
full: 9999px (pills, avatars)
```

### Shadows

```css
/* Minimal shadows for dark mode */
shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.3)
shadow-md: 0 4px 6px rgba(0, 0, 0, 0.4)
shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.5)
```

## Responsive Breakpoints

| Breakpoint | Width | Layout |
|------------|-------|--------|
| Mobile | < 640px | Single column, bottom nav |
| Tablet | 640px - 1024px | Collapsible sidebar |
| Desktop | > 1024px | Fixed sidebar, multi-column |

---

## Accessibility Notes

- All interactive elements must be keyboard accessible
- Focus states visible (purple outline)
- Color contrast ratio minimum 4.5:1
- ARIA labels for icons and non-text elements
- Form labels properly associated
- Error messages announced to screen readers
- Skip navigation link for keyboard users
